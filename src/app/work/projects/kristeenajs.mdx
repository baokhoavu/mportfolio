KristeenaJS demonstrates client-side e-commerce flows using SnipCart and a responsive UI built with SCSS and TypeScript.

The prototype highlights secure client-side interactions, streamlined checkout UX, and modular css patterns for quick theming.

## Key challenges

- Implementing secure client-side payment flows while maintaining UX simplicity.
- Creating an inventory and cart experience that performs well on mobile.

## Impact

- Produced a lightweight e-commerce prototype that improved conversion-ready UX.
- Demonstrated a rapid prototyping approach for client-facing donation and purchase flows.
