Online Check In is a full stack app built with Angular that integrates with Blackbaud Luminate to manage donations and event operations with robust server APIs.

Focus areas included secure API patterns, resilient offline-friendly workflows for event staff, and streamlined reporting for fundraising teams.

## Key challenges

- Securely integrating with Blackbaud's APIs while preserving user session integrity.
- Designing role-based workflows for volunteers and event staff under tight performance constraints.

## Impact

- Streamlined on-site check-in and donation reporting, reducing manual work during events.
- Enabled consistent fundraising flows tied to CRM records for accurate reporting.
