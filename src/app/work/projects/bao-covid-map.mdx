Bao's COVID-19 Tracking Map is a responsive web app built to visualize the global spread of coronavirus during the early pandemic. It features an interactive dashboard showing confirmed cases, recoveries, and deaths for every country, with historical data visualized through charts and a global map view. The frontend is built with Create React App, Mapbox, Recharts, and Styled Components, and pulls real-time data from public APIs. The app can be connected to MongoDB Atlas for scalable data storage and is deployed on Vercel for fast, reliable performance. The project emphasizes accessibility, performance, and clear data presentation, helping users stay informed with up-to-date pandemic statistics.


## Key challenges

- Integrating multiple third-party APIs with different formats and rate limits.
- Visualizing complex, global health data in a clear, interactive way.
- Ensuring responsive performance across devices and browsers.
- Managing scalable data storage with MongoDB Atlas and Vercel deployment.


## Impact

- Delivered a real-time, interactive dashboard for global COVID-19 data.
- Improved public accessibility to pandemic statistics through responsive maps and charts.
- Provided a scalable, open-source foundation for future health data visualization projects.
