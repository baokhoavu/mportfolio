Bao Covid Map fetches public Covid datasets and renders them as interactive charts and maps, helping users explore trends geographically.

The dashboard emphasizes performance and accessibility, with filterable views and clear callouts for regional anomalies.

## Key challenges

- Integrating multiple third-party APIs with differing rate limits and formats.
- Presenting complex geographical data in a clear, performant way.

## Impact

- Delivered an interactive dashboard that helped visualize regional trends and anomalies.
- Improved public accessibility to pandemic data through responsive visualizations.
